## 🚀 Getting Started - Installation

1. To deploy this project you must clone the  repository through the following link:

```
https://github.com/Sif03/fakestoreapi
```

2. Install all dependecies

```
npm i
```


3. Run the project!
```
npm run dev or npm run start
```
